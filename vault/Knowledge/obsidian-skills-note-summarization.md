# obsidian-skillsによるノート要約

要約ノートを作成・編集するときは `obsidian-markdown` を使い、Obsidian Flavored Markdownの作法に合わせる。

- YAMLフロントマターを書く。
- 元ノートや関連ノートは `[[wiki link]]` でつなぐ。
- 強調が必要な要約ではcalloutを使ってよい。
- 埋め込みは、元情報をその場で表示する価値がある場合だけ使う。

複数ノートを横断した一覧、絞り込み、集計ビューが必要な場合は `obsidian-bases` を使う。

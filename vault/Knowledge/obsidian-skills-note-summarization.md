# obsidian-skills Note Summarization

Use `obsidian-markdown` when creating or editing summary notes so summaries preserve Obsidian Flavored Markdown conventions:

- YAML frontmatter/properties
- `[[wikilinks]]` for source and related notes
- callouts when a summary needs emphasis
- embeds only when embedding the source adds value

Use `obsidian-bases` when the desired output is a structured index or filtered summary view across many notes.

# ゲームアートの方向性

- フィールド上のオブジェクトは、遠くから見ても種類が判別できるアイコン調を優先する。
- 細かな陰影や写実的な描き込みは避け、太い輪郭と大きな単色面で形を明快にする。
- 木、石、池、塔、バリスタ、柵、トンカチ、経験値オーブは、キャラクターより情報量を抑えて画面を見やすくする。
- フィールド上の障害物は無造作な座標へ置かず、一定間隔のグリッドセルへ配置する。
- 1セルにつき障害物は最大1個とし、塔周辺には障害物を置かない空白を確保する。

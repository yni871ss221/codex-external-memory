# AreaSurvivors History

AreaSurvivors の大きな作業履歴。次回開始時は通常 [[area-survivors-current]] を読むだけでよい。

## 2026-05-31 初期ゲームシステム

- 中央塔防衛、領地塗り、敵スポーン、経験値オーブ、レベルアップ、ゲームオーバー、ロビー、永続強化ツリーの初期実装。
- ナイト、アーチャー、メイジの3キャラクターと基本攻撃を実装。
- 防衛柵、バリスタ塔、建造演出、ダメージ表記を追加。
- Level-up画面のキーボード操作を整備。

## 2026-06-01 ドット絵・Tilemap・HUD初期整理

- キャラクター、敵、塔、バリスタ、柵、木、岩、池、弾、エフェクトをドット絵寄りの画像へ差し替え。
- Tile PaletteとTilemap構成へ移行し、草地、Paint、Object Tilemapを整理。
- セル基準の建造プレビュー、配置ghost、建造中、完成状態の見た目を整理。
- 右側塔ステータス、左下建造メニュー、選択/ホバー状態のUI色を調整。

## 2026-06-02 2.5D紙モデルと柵調整

- 斜め見下ろしの2.5D紙モデル表示へ移行。
- `PaperBillboard`、`PaperMeshVisual`、Quad Mesh表示を導入。
- 塗り床セル、グリッド、奥行き方向の見え方を調整。
- 横柵・縦柵のセル占有、画像寸法、Previewと完成物の一致を何度も調整。

## 2026-06-03 塔・バリスタ・柵の立体表現

- 中心塔とバリスタ塔を画像付き薄型3Dモデルへ移行。
- 柵について低ポリ3D化を試作し、後に画像Prefab/紙モデル寄りの扱いやすい構成へ戻した箇所がある。
- `YSort` と `sortPivotOffsetY` により、塔の根元基準で前後関係を調整。
- Unity Scene ViewをGame Cameraへ合わせるEditorメニューを追加。

## 2026-06-05 背景チャンクとランドマーク

- 草原の大きな背景チャンク画像を配置。
- 小石、草むら、花、むき出しの地面などを背景チャンクへ集約し、単独タイルの浮きや白線問題を回避する方針にした。
- 1/2/4/8セルの木・森・石・山ランドマークを追加。
- 旧バージョンの木、石、池オブジェクトを削除。
- ランドマーク配置は距離帯ごとの数をInspectorから調整できるようにした。

## 2026-06-06 経験値オーブと整理

- 経験値オーブを画像に置き換え、アウトラインを追加。
- 古い画像や未使用機能を整理。
- HUD左上のプレイヤー情報、武器、HP/XP、ステータス表示を再構成。

## 2026-06-07 資源採取と資源建造

- 木・石の資源所持数を導入。
- 建造可能数方式から資源消費方式へ移行。
- 木・石ランドマークを採取対象化し、青床接続時だけ採取可能にした。
- 採取ゲージ、斧/つるはし、採取ポップアップ、HUD資源表示を実装。
- 斧、つるはし、ハンマー画像をサンプルベースで加工して差し替え。
- 画像を割り当てる際はアウトラインを基本適用する方針を明文化。
- コミット: `9d8c5a0 Add resource harvesting and construction costs`

## 2026-06-08 ステータス設定化

- プレイヤー/武器ステータスの多くを `GameConfig` からInspector調整できるようにした。
- 永続強化の増加量、ラン中レベルアップ強化量、Knight攻撃範囲/Mage爆発範囲/攻撃間隔下限などを設定値化した。
- HUD左側のステータス欄に、復活時間、弾速、射程/範囲を追加表示した。
- UniCLI Compile `0 errors / 0 warnings`、PlayMode Errorログ `0件`。

## 2026-06-08 新規ステータス拡張

- `StatBlock` と `PlayerStats` を追加し、基礎値、永続強化、ラン中強化を合成して扱う構造にした。
- ノックバック、防御力、経験値獲得量、体力自動回復、作業速度、資源獲得量を追加した。
- ノックバックは敵のみ対象、経験値は小数蓄積、防御は0ダメージ表記あり、自動回復は `Health` に接続できる共通コンポーネント方式にした。
- 最大HP増加時は全回復せず、増加分だけ回復するように修正した。
- 作業速度を建造進捗、伐採、採掘へ反映し、資源獲得量を採取tickへ反映した。
- HUD左側に新規ステータス6項目をSceneオブジェクトとして追加し、スキルツリーにも永続強化ノードを追加した。
- UniCLI Compile `0 errors / 0 warnings`、PlayMode Errorログ `0件`。

## 2026-06-08〜10 敵進行・検証基盤・マップ表示

- 襲撃方角、スポーンフェーズ、エリート、ボス、トークンドロップ、ゲームクリア進行を追加した。
- オークとオークキングへ専用歩行アニメーションを適用した。
- 敵の障害物回避、袋小路を抑えるランドマーク配置、スタック復帰処理を改善した。
- 再現待ちを減らすため、座標、対象、時刻、固定Seed、Assertionを設定できるGameplayTest基盤を整備した。

- マップを縦方向へ拡張し、外周描画と進行不可境界を調整した。
- HUD重複時の透明化を追加し、レベルアップ画面は透明化対象外にした。
